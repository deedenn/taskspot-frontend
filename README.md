# TaskSpot – демо фронтенд

Простой фронтенд-прототип таск-менеджера **TaskSpot** (React + Ant Design):

- Лендинг по адресу `/` с описанием возможностей сервиса и кнопками «Войти» и «Регистрация».
- Вход по email и паролю (демо-пользователи).
- Регистрация нового пользователя (в памяти приложения).
- Роль администратора (видит все проекты и все задачи).
- Пользователи видят только проекты, владельцами или участниками которых они являются.
- Управление участниками проекта (добавлять/удалять может владелец проекта или админ) в отдельном боковом окне.
- Видимость задач: задачу видят только админ, создатель задачи, исполнитель и наблюдатели.
- Редактирование задач (только создатель задачи).
- Workflow завершения задачи:
  - исполнитель отмечает задачу как выполненную;
  - у инициатора появляется задача на подтверждение;
  - инициатор может подтвердить выполнение (статус «Закрыта») или отправить на доработку (статус «В работе»).
- Проекты, задачи и категории (категории привязаны к проекту).
- Канбан-доска с колонками по статусам и Drag & Drop.
- Представление задач списком (таблица) с сортировкой по сроку.
- Фильтр по названию/описанию, категориям и исполнителю.
- Модалки для создания задач и категорий.
- Демоданные хранятся в памяти (без backend).

## Демо-аккаунты

- Администратор: `admin@example.com` / `admin`
- Обычный пользователь: `anna@example.com` / `password`

## Запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите dev-сервер:

```bash
npm start
```

Приложение откроется по адресу http://localhost:3000.
